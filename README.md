# Landing Page - Pràctica Final AV2 (M09)

Aquest projecte consisteix en el desenvolupament d'una Landing Page interactiva, dissenyada com a pràctica final per a la UF2 del mòdul M09 del CFGS de Desenvolupament d'Aplicacions Web a l'ETP Xavier. L'objectiu principal és demostrar l'aplicació pràctica dels coneixements adquirits en disseny web, integració multimèdia i interactivitat d'interfícies.

## Descripció del Projecte

La pàgina web està estructurada per oferir una experiència d'usuari fluida, assegurant la compatibilitat entre dispositius mitjançant un disseny plenament responsiu. S'ha posat especial èmfasi en la coherència visual, l'accessibilitat, l'ús adequat d'etiquetes semàntiques i el tractament del contingut multimèdia.

## Característiques Tècniques i Implementació

* **Estructura i Semàntica:** Utilització estricta d'etiquetes semàntiques HTML5 (`<header>`, `<main>`, `<section>`, `<footer>`) per millorar el posicionament i l'accessibilitat del codi.

* **Disseny i Estilització Avançada:** Implementació d'un disseny responsiu basat en el framework Bootstrap 5.3.0. A més, s'ha aplicat CSS3 personalitzat per afegir components avançats d'interfície d'usuari, com ara efectes d'ombra profunda, transicions fluides, interaccions de tipus "hover" i formularis amb etiquetes flotants (floating labels) per millorar la usabilitat.

* **Contingut Multimèdia:**
  * **Galeria:** Integració d'un carrusel interactiu d'imatges optimitzades.
  * **Vídeo i Àudio:** Ús de les etiquetes natives `<video>` i `<audio>` d'HTML5, combinant fonts externes i arxius locals per garantir una correcta reproductibilitat i rendiment als navegadors moderns.

* **Interactivitat i Validació de Dades:** Desenvolupament d'un script en JavaScript (Vanilla JS) que captura els esdeveniments del formulari de contacte. Aquest inclou un sistema de validació prèvia, simulació d'estats de càrrega asíncrona (modificant el DOM del botó d'enviament) i la integració de la llibreria externa **SweetAlert2** per proporcionar un feedback visual clar i professional a l'usuari final.

## Tecnologies Utilitzades

* HTML5
* CSS3 (Estils personalitzats)
* JavaScript (ES6)
* Bootstrap 5.3.0 (Framework CSS)
* SweetAlert2 11 (Gestió d'alertes modals)
* FontAwesome 6.4.0 (Iconografia vectorial)
