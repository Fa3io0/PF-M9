# Landing Page - Pràctica Final AV2 (M09)

Aquest projecte és una Landing Page desenvolupada com a pràctica final de la UF2 del mòdul M09 de Desenvolupament d'Aplicacions Web a l'ETP Xavier.

## 🚀 Característiques i Tecnologies utilitzades

* **Estructura HTML5 Semàntica:** Ús correcte d'etiquetes com `<header>`, `<main>`, `<section>` i `<footer>`.
* **Disseny Responsiu (CSS3 & Bootstrap 5):** Adaptable a qualsevol dispositiu mòbil o d'escriptori. Estils avançats i personalitzats.
* **Interactivitat (JavaScript):**
    * Galeria d'imatges interactiva implementada amb el Carrousel de Bootstrap.
    * Integració de reproductors natius de vídeo (`<video>`) i àudio (`<audio>`).
* **Formulari de Contacte:** Simulació d'enviament amb validació i feedback visual atractiu utilitzant la llibreria **SweetAlert2**.
* **Xarxes Socials:** Botons integrats amb icones de FontAwesome per a una fàcil navegació.
